# Songbook XP Support

A viewer for `.songbook` archives with offline playback. This page covers common questions and how to get in touch.

## Getting started

### How do I open a songbook?

Tap any `.songbook` file on your phone — from email, your file manager, a download or another app — and choose Songbook XP from the "Open with" prompt. The app imports the book into local storage and opens it.

### Where do I get `.songbook` files?

The format is open and documented. You can:

- Build your own — see the [file format documentation](https://github.com/vbullinger/SongbookXP#songbook-file-format) in the repository README.
- Receive one from someone via email, AirDrop, Drive or any sharing tool.
- Use a file produced by the upstream Swift app — both apps share the same archive format.

### Why doesn't Songbook XP request internet permission on Android?

Songbook XP runs fully offline. The Android release build explicitly omits the `INTERNET` permission; you can verify this in your system settings.

### How do I change the font?

Tap the **Aa** button at the top of any page. Two accessibility-optimized typefaces are available: APHont and Atkinson Hyperlegible.

### What happens to my data if I uninstall?

Everything stored locally — imported books, preferences, open-page state — is removed when you uninstall the app. No cloud backup, no server-side copy.
